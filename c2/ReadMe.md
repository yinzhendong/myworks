#### 原网站拷贝文件到本地
1. 原文章保存为本地的HTMl文件，文件名为文章标题
2. 原网站可以从标题取目录结构信息
3. 文章中的图片保存在本地
4. 修改文章中的图片地址链接为本地图片
5. 文章中的附件需要保存到本地

#### 遇到的问题
1. 文章中有些图片由外链
2. Ubuntu系统下保存后的图片提示格式不正确，但是在windows下打开正常

#### 原网站说明
1. 原网站文章链接形式为
`http://www.c2.org.cn/index.php?m=content&c=index&a=show&catid=126&id=868`
2. 链接中的catid其实没什么用，只要指定一个存在的就可以把所有的都取出来，目前文章编号到909
